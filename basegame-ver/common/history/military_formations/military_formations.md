NB: When editing naval formations keep the following in mind:

- The current ship distribution at game start has ships assigned to their historical naval formatins and locations. 
- Ship name allocation is based off read order, so changing the order or number of ships generated in the military_formations files must be done in conjunction with changes to the ship_name_definitions files.
- The ship_name_definitions .md file contains a list of the intended naval OOB in 1836.
